6. How It Works

  1. Script loads and initializes automatically
  2. Monitors page navigation via body ID changes
  3. Tracks each unique calculator used in localStorage
  4. Checks for newly-earned badges after each calculator use
  5. Displays toast notification when badges are unlocked
  6. Trophy button shows badge count
  7. Clicking trophy opens achievement panel
  8. Users can share their progress via native share or clipboard
